1. Standard Use (Basic Gallery)

To convert a group of images and/or videos into a working, swipeable gallery with lightbox functionality, simply wrap them in a <div> with the required class.

Action Required:

Add the class vetrom-gallery to the parent div tag that contains your media.

Example 1: Images & Standard Video

<div class="vetrom-gallery">
    <img src="./image1.png" alt="Slide 1">
    <img src="./image2.png" alt="Slide 2">
    <video src="./test.mp4" controls></video>
</div>

2. Integration with Vetrom Player

You can combine the Gallery with the Vetrom Player for a premium video experience inside your carousel. This requires both scripts to be loaded.

Note: Ensure you have also included player.js in your head/body for this to work.

Example 2: Images & Vetrom Player Video

<div class="vetrom-gallery">
    <img src="./image3.png" alt="Slide 3">
    <img src="./image4.png" alt="Slide 4">
    <!-- Video with Vetrom Player Class -->
    <video src="./test.mp4" class="vetrom-media-selected"></video> 
</div>

3. Standalone Vetrom Player

You can also use the Vetrom Player independently, without the gallery carousel. This is useful for single featured videos on a page.

Note: Ensure you have also included player.js in your head/body for this to work.

Example 3: Standalone Player

<!-- Standalone Player Wrapper -->
    <video src="./test.mp4" class="vetrom-media-selected"></video>

4. Advanced Use (Multiple Galleries)

If you need multiple independent galleries on the same page, you can optionally append a unique number after the class name (e.g., :1, :2).

Example: Multiple Unique IDs

<!-- Gallery #1 -->
<div class="vetrom-gallery:1">
    <img src="./image5.png">
</div>

<!-- Gallery #2 -->
<div class="vetrom-gallery:2">
    <img src="./image6.png">
</div>

5. Theme Customization

The gallery automatically adapts to your website's background color. However, you can enforce a specific color scheme using CSS variables.

CSS Example:

:root {
    /* Gallery background color (slides) */
    --vetrom-bg: #222222; 
    
    /* Text/Icon color for controls */
    --vetrom-text: #ffffff; 
    
    /* Background color for controls (buttons) */
    --vetrom-accent: rgba(255, 255, 255, 0.4); 
}

✅ Idiot-Proof How-To Use: 3 Simple Steps

Follow these steps to ensure the gallery works correctly in your HTML file.

  1. Copy the Setup Script (Always Required):

    Copy the line below and paste it somewhere within the <head>...</head> or <body>...</body> tags of your HTML document.

    <script src="https://api.mahdiyasser.site/gallery/v1/gallery.js"></script>
  2. Create the Container Div:

    Create a <div> element where you want your gallery to appear and give it the specific class name.

    <div class="vetrom-gallery">
    ...
</div>
  3. Add Your Media:

    Simply paste your <img> or <video> tags inside that div. The script will handle the rest!

    <div class="vetrom-gallery">
    <img src="image.jpg">
    <img src="photo.png">
    <video src="movie.mp4"></video>
</div>
