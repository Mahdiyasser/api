ðŸŽ¨ 2. Theme Customization

V2 introduces powerful theming with the new class format:

vetrom-gallery-{theme}-{color}

Available Themes:

  • auto - Automatically detects page background (default)
  • dark - Force dark theme
  • light - Force light theme

Color Options:

  • Named colors: red, blue, green, purple, orange, pink, etc.
  • Hex values: #FF5733, #3498db
  • RGB: rgb(255,87,51)
  • RGBA: rgba(52,152,219,0.8)

Theme Examples:

<!-- Dark theme with blue accent -->
<div class="vetrom-gallery-dark-blue" id="gallery2">
    <img src="./image3.png">
    <img src="./image4.png">
</div>

<!-- Light theme with custom hex color -->
<div class="vetrom-gallery-light-#FF5733" id="gallery3">
    <img src="./image5.png">
    <video src="./test.mp4" controls></video>
</div>

<!-- Auto theme with RGB color -->
<div class="vetrom-gallery-auto-rgb(52,152,219)" id="gallery4">
    <img src="./image6.png">
</div>

âš¡ 5. Dynamic Content Support (NEW!)

V2 uses a MutationObserver to watch the DOM 24/7. Galleries added dynamically will automatically initialize!

Example: Add Gallery Dynamically

<button onclick="addGallery()">Add Gallery Dynamically</button>
<div id="dynamic-container"></div>

<script>
function addGallery() {
    const container = document.getElementById('dynamic-container');
    const gallery = document.createElement('div');
    gallery.className = 'vetrom-gallery-dark-cyan';
    gallery.id = 'dynamic-gallery-' + Date.now();
    gallery.innerHTML = `
        <img src="./image5.png">
        <img src="./image6.png">
        <video src="./test.mp4" controls></video>
    `;
    container.appendChild(gallery);
}
</script>
