Vetrom Player V1 — The Ultimate MediaPlayer API

⚠ Required Setup: The API script must be included in your <head> or <body> to enable the player functionality:

<script src="https://api.mahdiyasser.site/player/v1/player.js"></script>

1. Standard Use (Basic Player)

For the simplest integration, apply the required class to any <video> element. The Vetrom Player will automatically initialize.

Action Required:

Add the class vetrom-media-selected to your <video> tag.

Example (Copy-Paste Ready):

<video src="video.mp4" class="vetrom-media-selected"></video>

2. Advanced Use (Adding Chapters/Checkpoints)

To include navigational chapters on the video timeline, you must identify the video using an ID and define the checkpoints in a <meta> tag within the document's <head>.

Step A: Set the Video Target

Assign a unique ID to your target video element (e.g., id="the-test-video-advanced").

Step B: Define Checkpoints

Add the required <meta> tag inside the <head> of your HTML document.

The Points Format is: MM:SS-Chapter Name. Separate multiple chapters with a comma (,).

Complete Example:

In the <head> (Chapter Definition):
<meta name="vetrom-checkpoint" points="00:5-good part,00:10-better" target="the-test-video-advanced">
In the <body> (Video Element):
<video src="test.mp4" class="vetrom-media-selected" id="the-test-video-advanced"></video>

✅ Idiot-Proof How-To Use: 3 Simple Steps

Follow these steps to ensure the player works correctly in your HTML file.

  1. Copy the Setup Script (Always Required):

    Copy the line below and paste it somewhere within the <head>...</head> or <body>...</body> tags of your HTML document. The <head> is usually the best place.

    <script src="https://api.mahdiyasser.site/player/v1/player.js"></script>
  2. Add the Class to Your Video (Required for Player):

    Find the <video> tag you want to upgrade and add the class class="vetrom-media-selected" to it. This is what turns a regular video into the Vetrom Player.

    <video src="your-video-file.mp4" class="vetrom-media-selected"></video>
  3. (Optional) Add Chapters:

    If you want chapters, you need two things:

    • Give your video an ID: Change your video tag to include a unique ID, like id="my-unique-video".
    • Add the Meta Tag: Paste the chapter definition (the <meta> tag) inside the <head>. Make sure the target="..." attribute matches the ID you just set!
    <head>
    ...
    <meta name="vetrom-checkpoint" points="00:15-Intro,01:30-Conclusion" target="my-unique-video">
</head>
<body>
    ...
    <video src="my-video.mp4" class="vetrom-media-selected" id="my-unique-video"></video>
</body>
