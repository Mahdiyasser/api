1. Basic Usage

Simply add the class vetrom-media to any <video> element. This uses the default theme (baby blue, white, and gold).

Example: Basic Player

<video src="video.mp4" class="vetrom-media"></video>

3. Chapters (Time Markers)

Add navigational chapters using the video's id attribute.

Format: Chapter Name,HH:MM:SS-Chapter Name,HH:MM:SS-...

‚ö†Ô∏è Important: Time format must be HH:MM:SS (hours:minutes:seconds). Use 00:00:05 for 5 seconds, not 00:05.

Example: Player with Chapters

<video 
  src="video.mp4" 
  class="vetrom-media-dark-purple"
  id="Introduction,00:00:00-Main Content,00:00:10-Conclusion,00:00:20">
</video>

4. Secure Mode (Copyright Protection)

Enable secure mode to show a custom context menu on right-click/long-press. Add secure to the class name.

Example: Secure Player

<video src="video.mp4" class="vetrom-media-secure-dark-red"></video>

5. Multiple Quality Sources

Provide multiple quality options using standard HTML5 <source> elements with label and res attributes.

Example: Multi-Quality Player

<video class="vetrom-media-dark-cyan">
  <source src="video-1080p.mp4" type="video/mp4" label="1080p" res="1080">
  <source src="video-720p.mp4" type="video/mp4" label="720p" res="720">
  <source src="video-480p.mp4" type="video/mp4" label="480p" res="480">
</video>

6. Subtitles/Captions

Add subtitles using standard HTML5 <track> elements. Add default attribute to enable automatically.

Example: Player with Subtitles

<video class="vetrom-media-light-green">
  <source src="video.mp4" type="video/mp4">
  <track kind="subtitles" src="subs_en.vtt" srclang="en" label="English" default>
